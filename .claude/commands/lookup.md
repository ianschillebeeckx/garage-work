## Tips

- Building codes vary by jurisdiction. Always identify which code cycle the jurisdiction has adopted (e.g., NEC 2020 vs 2023, CBC 2022, CPC 2022).
- City municipal codes often have local amendments that override state/national codes.
- "Over the counter" permits vs. plan-check permits have very different timelines.
- For gray water, California has specific statewide standards but cities can be more restrictive.
