# Decision: Drywall Requirements by Wall/Ceiling

**Date:** 2026-03-25
**Source:** California Residential Code R302.6, Table R302.6
**URL:** https://up.codes/s/dwelling-garage-and-or-carport-fire-separation

## Code: CRC R302.6 — Dwelling-Garage Fire Separation

Table R302.6 specifies:

| Separation | Required Material |
|---|---|
| Garage from residence and attics | Min. 1/2" gypsum board, applied to garage side |
| Habitable rooms above garage/carport | Min. 5/8" Type X gypsum board |
| Structural supports for floor/ceiling assemblies used for separation | Min. 1/2" gypsum board |
| Garage within 3ft of other dwelling on same lot | Min. 1/2" gypsum board on interior side of exterior walls |

## Application to This Garage

| Surface | What's on the other side | Drywall required? | Spec |
|---------|--------------------------|-------------------|------|
| **Ceiling** | 2nd floor living space (habitable rooms above) | **YES** | 5/8" Type X gypsum board |
| **West wall** | 1st floor living space (residence) | **YES** | 1/2" gypsum board on garage side |
| **North wall (target wall)** | Exterior / street | **NO** | Not separating from residence or attic — not listed in Table R302.6. Plywood can remain exposed. |
| **East wall** | Exterior | **NO** | Same as north wall |
| **South wall** | Exterior / door | **NO** | Confirm nothing habitable on the other side |

## Key Finding

The north (target) wall does not require drywall under CRC R302.6. It is an exterior wall with no habitable space on the other side. Plywood sheathing can remain as the finished interior surface.

The "within 3ft of other dwelling" row does not apply — the house is detached (RH-1(D) zoning) and not within 3ft of a neighboring dwelling.

## What This Means for the Project

- **North wall (target):** Plywood only — no drywall needed. Can serve as workshop wall surface.
- **Ceiling:** Must install 5/8" Type X drywall (already in project plan, Phase 5).
- **West wall:** Must have 1/2" drywall on garage side. Currently partially drywalled with tile — confirm existing coverage is adequate or plan to complete.
- Structural supports (beams/columns supporting the floor above) also need 1/2" gypsum if not already covered.
