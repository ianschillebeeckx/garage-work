# Jurisdiction

- **Zoning:** RH-1(D) — Residential House, One Family, Detached
- **City/County:** San Francisco, CA (city and county are the same entity)
- **Permitting Authority:** San Francisco Department of Building Inspection (SFDBI)
- **Website:** sfdbi.org
- **Codes Adopted:** California Building Code (CBC), California Electrical Code (CEC), California Plumbing Code (CPC), with SF local amendments
- **Permit Portal:** SF Planning + DBI online permit portal (sfgov.org)

## SF-Specific Notes

- SF is its own permitting authority (not a county agency)
- DBI handles building, electrical, and plumbing permits separately
- Electrical and plumbing permits are typically "over the counter" (no plan review) for residential work of this scale
- SF allows homeowners to pull permits for their own owner-occupied residence
- SF has adopted the California Green Building Standards Code (CALGreen), relevant for gray water
