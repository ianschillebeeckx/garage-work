# Plumbing Permit — San Francisco DBI

## Scope of Work

- Reroute washer supply (hot + cold) from external/surface-mounted to in-wall
- Install recessed washer outlet box with shutoff valves
- Install washer drain standpipe (2" typically)
- Bath gray water diversion (3-way valve, subsurface irrigation)
- Remove or cap existing garage sink connections (if applicable)

## Do I Need a Permit?

**Yes for the washer reroute and bath gray water diversion.**

The washer laundry-to-landscape gray water system does NOT need a permit (see `decisions/permits-needed-summary.md`).

## SF DBI Permit Process

**Authority:** San Francisco Department of Building Inspection (SFDBI)
**Type:** Plumbing permit (separate from electrical/building)
**Process:** Over-the-counter for homeowner — in person only

### Steps

- [ ] Complete the **Plumbing Permit Application Worksheet** (blue form — available at sf.gov or DBI)
- [ ] Mark "homeowner" on the application
- [ ] Bring application to DBI, 49 South Van Ness Ave, 4th floor, Inspection Services counter (plumbing division)
- [ ] Bring photo ID proving you are the owner
- [ ] Receive approval and permit number (same day for OTC)
- [ ] Do the work
- [ ] Schedule rough-in inspection BEFORE covering any pipes (call or use DBI online scheduling)
- [ ] Pass rough-in inspection
- [ ] Close walls
- [ ] Schedule final inspection
- [ ] Pass final inspection

### Scope Description (for application)

> Reroute washing machine supply lines (hot and cold) from exterior surface-mounted to in-wall. Install recessed washer outlet box with shutoff valves and 2" drain standpipe with P-trap. Install bath/shower gray water diversion with 3-way valve and subsurface landscape irrigation. Remove/cap existing garage sink connections.

## Homeowner Pull Notes

- SF allows homeowners of owner-occupied **single-family dwellings** to pull their own plumbing permits.
- Homeowners must apply **in person** — cannot use the online contractor portal.
- Same eligibility rules as electrical: must be a stand-alone single-family dwelling, bring photo ID matching property owner on record.

## Notes

- Washer drain standpipe height is code-regulated (typically 18"–42" above floor)
- Standpipe must connect to a proper trap (P-trap)
- If rerouting drain, maintain proper slope (1/4" per foot minimum)
- All work must be inspected before covering with plywood/drywall
