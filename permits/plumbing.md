# Plumbing Permit

## Scope of Work

- Reroute washer supply (hot + cold) from external/surface-mounted to in-wall
- Install recessed washer outlet box with shutoff valves
- Install washer drain standpipe (2" typically)
- Remove or cap existing garage sink connections (if applicable)

## Permit Status

- [ ] Determine local permitting authority
- [ ] Review permit application requirements
- [ ] Submit application
- [ ] Receive approval
- [ ] Schedule rough-in inspection (before drywall)
- [ ] Pass rough-in inspection
- [ ] Schedule final inspection
- [ ] Pass final inspection

## Notes

- Washer drain standpipe height is code-regulated (typically 18"–42" above floor)
- Standpipe must connect to a proper trap (P-trap)
- If rerouting drain, maintain proper slope (1/4" per foot minimum)
